4 · Dynamic Behaviour — When Each Indicator Is Shown
Scroll position ▲ top indicator ▼ bottom indicator
Content fits (no overflow) ✗ hidden ✗ hidden
At top (scrollTop ≤ 4px) ✗ hidden ✓ visible
Middle (scrolled partway) ✓ visible ✓ visible
At bottom (fully scrolled) ✓ visible ✗ hidden
Panel collapsed / hidden ✗ n/a ✗ n/a
Implementation hook: A custom useScrollIndicators(ref) hook returns { showTop, showBottom } booleans.
Attach a scroll listener + ResizeObserver on the container ref. Re-check on content changes (location prop, log prop).
The hook is reusable across CollapsibleRoomDescription, CombatLog, Inventory, NpcChatPanel, and any future scrollable panel — no per-component code duplication.
